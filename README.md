#  Hola, soy Dairon Manzo

## QA Engineer Jr. | API Testing & Test Automation

Soy QA Engineer Jr. con experiencia práctica en pruebas manuales, Web y API, con un enfoque especial en la validación de servicios REST utilizando Postman y en la automatización básica de pruebas con Python y Pytest.

Disfruto analizar el comportamiento de las APIs, diseñar casos de prueba, validar reglas de negocio, verificar códigos de respuesta HTTP y documentar hallazgos de forma clara y estructurada. También he desarrollado proyectos de automatización utilizando Python, Selenium WebDriver y el patrón Page Object Model (POM), fortaleciendo mis conocimientos en pruebas funcionales automatizadas.

Actualmente estoy construyendo proyectos personales inspirados en escenarios reales de la industria para demostrar mi forma de pensar como QA Engineer. Mi objetivo es crear casos de estudio bien documentados que muestren no solo el uso de herramientas como Postman y Python, sino también mi capacidad para analizar requisitos, identificar riesgos, diseñar estrategias de prueba y comunicar resultados de manera profesional.
